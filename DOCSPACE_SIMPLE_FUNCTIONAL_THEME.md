# DocSpace - Tema simples e funcional

Esta versão mantém a lógica original do projeto e aplica uma camada visual final em CSS, inspirada em ferramentas web simples como Smallpdf.

Objetivo:
- interface clara;
- topo limpo;
- cards simples;
- menos sombras e animações;
- layout funcional em desktop e mobile;
- preservar `.is-hidden` e os controles JavaScript originais.

Arquivos alterados:
- `style.css`
- `dist/style.css`

A camada foi adicionada no final do CSS para evitar quebrar HTML/JS existente.

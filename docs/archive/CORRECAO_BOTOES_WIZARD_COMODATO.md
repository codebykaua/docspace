# Correção dos botões do wizard

- Corrigido contrato de comodato sem botão Continuar.
- A navegação Voltar/Continuar agora é criada automaticamente quando não existir.
- Barra de etapas também é criada quando ausente.
- Botões Gerar Word, Gerar PDF e Limpar campos aparecem somente na última etapa.
- O botão PDF é forçado a aparecer na etapa final.

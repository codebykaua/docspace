# DocSpace — Layout Global Simples e Funcional

Aplicação visual global inspirada em ferramentas simples tipo Smallpdf, mantendo a identidade DocSpace.

## O que foi aplicado

- Tema claro global, mais simples e funcional.
- Header superior limpo, sem excesso de brilho/animação.
- Cards, botões, inputs, filtros, modais e painéis padronizados.
- Dashboard, documentos, categorias, favoritos, recentes, downloads, IA/admin/configurações, perfil, PDF tools e modais receberam camada visual única.
- Preservado o controle original do projeto por `.is-hidden`, sem trocar a lógica JS.
- Mantidos `style.css` e `dist/style.css` sincronizados.

## Observação

Esta alteração é uma camada final de CSS para padronização visual. A lógica de documentos, login, permissões, geração de arquivos, admin e IA não foi alterada.

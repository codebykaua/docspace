# DocSpace Layout Simples Funcional V3

Correção aplicada após a versão anterior ficar pesada e com um painel escuro indevido.

Esta versão preserva a lógica original do projeto e aplica apenas uma camada visual global mais simples:

- fundo branco;
- header simples;
- navegação horizontal limpa;
- cards claros;
- dashboard sem bloco preto gigante;
- menos sombras, brilho e animações;
- compatibilidade com `.is-hidden`;
- sincronizado em `style.css` e `dist/style.css`.
